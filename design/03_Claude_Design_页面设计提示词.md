# 星座百科 · Claude Design 页面设计提示词

> 用法：先把【0. 设计系统总纲】粘给 Claude Design 建立统一基调，再逐页粘贴对应提示词。每条提示词都自包含，可单独使用。产品为中英双语，提示词中已给出双语示例文案。

---

## 设计方向（先读，建立共识）

**一句话定位**：一个面向全球的星座性格分析站——读起来像一本高级占星杂志，用起来像一台精密的人格诊断仪。

**刻意回避**（这些是当前 AI 生成占星页的套路，不要用）：满屏紫色霓虹渐变、闪烁星星贴图、神秘主义剪贴画、水晶球 emoji、廉价光晕。我们要的是克制、编辑感、可信度。

**记忆点（签名元素）**：**六维能量罗盘**——一个把用户"火/土/风/水 + 内显↔外显 + 守序↔混沌"六维得分画成星图式雷达的圆形图。它既是功能核心（展示维度得分），又是视觉签名，还是分享卡的主角。整站的"星座味"集中花在它身上，其余保持安静。

## 0. 设计系统总纲（第一条粘贴）

```
为一个名为「星座百科 / Astro Persona」的全球中英双语星座性格分析网站建立设计系统。
基调：高级占星杂志 × 精密人格诊断仪。克制、编辑感、可信，绝不廉价神秘风。

【配色】深色为主，金属点缀，四象各有专属色光：
- 夜墨 Midnight Ink  #0E1530（主背景，深蓝黑而非纯黑）
- 深空 Deep Space    #1B2452（卡片/分区背景）
- 骨白 Bone          #F4EFE6（深色上的文字 / 浅色区背景）
- 星金 Star Gold     #C9A24B（唯一金属强调色，用于强调与签名，克制使用）
- 火 Ember           #E2683C
- 土 Sage            #8A9A5B
- 风 Mist            #9FC1D4
- 水 Tide            #2E7C8A
四象色仅用于能量罗盘、维度标签与数据，不要泛滥到整页。

【字体】双语配对：
- 拉丁标题：一款高对比度、有文学气质的衬线（如 Fraunces / Cormorant 风格），克制使用，仅大标题
- 中文标题：思源宋体 / Noto Serif SC（与拉丁衬线气质呼应）
- 正文：人文无衬线（拉丁 Inter，中文 Noto Sans SC）
- 数据/分数：等宽字体（如 IBM Plex Mono），用于维度得分、百分比——强化"被测量过"的诊断感

【签名元素】六维能量罗盘：圆形星图式雷达，六个轴=火/土/风/水/内显-外显/守序-混沌，
轴端以四象色点亮，用户得分连成一个发光多边形，置于夜墨背景上，星金描边。
这是全站记忆点，必须精致；其余元素保持安静、留白充足、字号敢大。

【其它】移动优先、键盘可达、可见焦点、尊重 prefers-reduced-motion；
动效克制（页面载入的一次编排、滚动揭示、罗盘绘制动画即可，不要到处闪）。
所有界面文案中英双语并存的页面需自然切换，不做机翻腔。
```

## 1. 首页 / Landing（`/zh` `/en`）

```
设计「星座百科」首页落地页，沿用既定设计系统（夜墨底、星金点缀、能量罗盘为签名）。
目标：让访客 10 秒内理解"这不是普通运势站"，并走向两条主路径：读星座百科 / 做性格测试。

英雄区（hero）：不要用"大数字+小标签+渐变"的模板答案。改用一句有态度的衬线大标题 +
一个缓缓绘制的能量罗盘剪影作为视觉主角。
双语示例文案：
  中文："星座是起点，不是终点。"  副标题："测一测你被星座框住了多少。"
  English: "Your sign is a starting point, not a verdict."  Sub: "Find out how far you've drifted from it."
两个主行动按钮："开始性格测试 / Take the test"（主，星金）、"浏览十二星座 / Explore the 12 signs"（次，描边）。

下方分区（保持安静、克制）：
- 四象一览：火/土/风/水四个色块卡，各配一句性格速写，点击进入对应星座群。
- "异变"概念预告：用一两句话+一个偏离的罗盘示意，埋下"你可能不典型"的钩子。
- 一行底部免责声明（小字）："娱乐与自我探索用途，非科学结论 / For entertainment and self-reflection, not scientific advice."

语言切换放在顶栏显眼处（中 / EN）。响应式：移动端 hero 文字优先、罗盘置于其下。
```

## 2. 星座百科：列表 + 详情

```
设计「十二星座百科」的列表页与单星座详情页，沿用设计系统。

列表页：按四象（火/土/风/水）分组的 12 星座卡片墙，每卡含星座符号、名称（中英）、
日期范围、一句性格关键词。四象用各自色光区分但整体仍以夜墨为底，星金为强调。
顶部一句编辑式导语，而非堆砌关键词。

详情页（如 白羊座 / Aries）：杂志式排版，敢留白、字号敢大。
模块顺序建议：
1) 头图区：星座符号 + 名称（衬线大标题，中英并置）+ 日期/守护星/元素/三态（用等宽小字像档案标签）
2) 性格速写：2–3 段编辑式正文（优点与缺点都讲，不吹捧）
3) 该星座的"共识基线罗盘"：用能量罗盘展示这个星座的典型六维画像
4) 爱情 / 事业 倾向：两栏短文
5) 相性速览：与其它星座的简明相性（高/中/低），用四象色编码
6) 行动召唤："测一测我是不是典型的白羊 / Am I a typical Aries?" → 进入测试
底部免责声明小字（双语）。SEO 友好：清晰层级标题、语义结构。
响应式：详情页移动端单列纵向，相性表转为可横滑。
```

## 3. 性格测试：问卷流程

```
设计「性格测试」答题流程页，沿用设计系统。氛围：像在做一次安静而专注的诊断，而非娱乐抽卡。

进入前：一屏选择年龄段（如 <18 / 18–25 / 26–35 / 36–45 / 46+）与题量档位（快测约12题 / 完整约20题），
说明"不会询问你的精确生日"。双语。

答题界面：一次一题（情境化单选/二选一），顶部细长进度条 + "第 n / N 题"等宽计数。
题面用衬线、选项用无衬线卡片，hover/选中以星金细描边反馈。支持"上一题"。
背景极简夜墨，让注意力集中在题目；可有极淡的星点纹理但不可喧宾夺主。
双语示例题：
  中文："周末空出来了，你更想——"  A) 临时起意出门冒险  B) 在家把计划中的事做完
  English: "A free weekend opens up. You'd rather—"  A) Go somewhere on a whim  B) Finish what you'd planned at home
最后一题完成后：一个"正在生成你的能量画像 / Reading your chart…"的过渡动画（罗盘逐步绘制），
再进入结果页。响应式：移动端选项大按钮、单手可达。
```

## 4. 测试结果页（含异变 + 分享卡）

```
设计「性格测试结果页」，沿用设计系统。这是全站的高光页，能量罗盘是绝对主角。

布局：
1) 顶部：用户的能量罗盘（大、发光、星金描边），中心是原创原型名（衬线大字，如「炽焰先锋 / The Ember Vanguard」）
2) 六维得分：用等宽字体逐维列出分数与简短标签（火 82 / 土 40 …），点出主导维与缺失维——
   这一段是"可信度"的关键，要让人看到结论有数据依据，而不是泛泛而谈。
3) 性格关键词：3–5 个标签，四象色编码。
4) 与星座的关系：
   - 若"典型"：展示用户罗盘与星座基线罗盘的贴合，文案偏印证。
   - 若"异变"：把两张罗盘叠放并高亮偏离的轴，给出分档（微异/显异/极异）徽章，
     文案要"燃"且温暖：
     中文："你已经走出了星座给的剧本。这一切，都是你一路经历换来的。"
     English: "You've outgrown the script your sign handed you—every bit of it earned on the way here."
5) 行动召唤："看看谁和你最合 / See who fits you best"（进入情感匹配）。

【分享卡】单独设计一张竖版分享卡片（适合社媒）：夜墨底、居中能量罗盘、原型名、3 个关键词、
异变徽章（若有）、底部品牌水印与极小网址。要"好看到想截图"，但保持克制与高级感，不要堆装饰。
提供"保存图片 / Save image"与"分享 / Share"按钮。
响应式：移动端罗盘置顶、得分纵向排列。
```

## 5. 情感匹配：输入页（含理想型）

```
设计「情感匹配」输入页，沿用设计系统。前提：用户已完成性格测试（已有能量画像）。

结构：
1) 简短说明：匹配 = 传统星座相性 + 你的答题人格 + （可选）你的理想型，越填越准。双语。
2) 一个清晰的可选开关："要不要告诉我们你的理想型？/ Want to describe your ideal match?"
   - 关：直接基于用户人格与星座相性匹配。
   - 开：展开一个轻量的理想型描述器——用与测试同维度的几组对照滑块/二选一
     （如 更冒险 ↔ 更安稳、更外放 ↔ 更内敛），生成理想型向量。强调"几秒钟即可"。
3) 主按钮："开始匹配 / Find my matches"（星金）。
视觉：延续诊断仪气质，理想型描述器用罗盘的四象色给滑块上色，呼应签名元素。
响应式：滑块在移动端易拖动、有数值/标签反馈。
```

## 6. 匹配结果页

```
设计「情感匹配结果页」，沿用设计系统。

布局：
1) 顶部：一句结论 + 用户与最佳匹配星座的"双罗盘"叠图（用户实线、对方虚线），直观看相似与互补。
2) 推荐榜：Top 3–5 星座卡片，按匹配分降序。每卡含：
   - 星座符号 + 名称（中英）+ 匹配分（等宽大字，如 88%）
   - 一句解释"为什么合"：点明元素关系 + 命中的相似维/互补维 +（若填了理想型）理想型契合度
     中文示例："土象同频，你的'稳'恰好接住对方的'冲'；与你的理想型高度吻合。"
     English: "Earthy resonance—your steadiness catches their fire, and it closely matches your ideal."
   - 四象色编码的小标记区分相似/互补。
3) 若用户本人为异变：加一行说明"已为你弱化星座、强化真实人格来匹配 /
   We weighted your real personality over your sign."
4) 行动召唤："生成匹配卡分享 / Share these matches" + "重新测 / Retake test"。
视觉克制，重点在双罗盘叠图与等宽匹配分的"诊断感"。响应式：榜单移动端单列。
```

## 附：给 Claude Design 的通用提醒（可附在任意一条之后）

```
- 把"星座味"集中在能量罗盘上，其余保持安静、留白充足、字号敢大。
- 强调色只用星金，四象色仅用于罗盘/维度/数据，避免全页花哨。
- 文案用主动语态、句子大小写、无废话；中英各自地道，不要机翻腔。
- 空状态与错误状态也要给方向，而非卖弄气氛。
- 守住质量底线：响应式到移动端、可见键盘焦点、尊重 reduced-motion。
```
